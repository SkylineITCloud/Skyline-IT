# Skyline IT

The official single-page website for **Skyline IT**, a South African holding company building brands across technology, IoT, electronics, fashion, and education.

The site presents the Skyline IT story, its portfolio companies, core values, industry focus, and contact information.

## Live site structure

```text
Skyline IT/
├── index.html       # Complete responsive website: markup, styles, and interactions
├── images/
│   └── Logo.png     # Skyline IT brand logo and favicon
└── README.md        # Project documentation
```

- **Mzansi Connect** — Affordable IoT products for South African homes, small businesses, utilities, and fleets.
- **Circuit Forge Technologies** — Electronics engineering, PCB design, embedded systems, firmware, and hardware prototyping.

- **StudySync** — A study-group management and real-time collaboration platform for students.

## Running the website

This project has no build step or package dependencies.

1. Open `index.html` in any modern web browser.
2. For the most reliable local testing experience, serve the folder with a local web server, then open the provided local address in a browser.

For example, if Python is installed:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customising content

All site content, styling, and JavaScript are contained in `index.html`.

- Update company descriptions, services, or founder information directly in the relevant HTML sections.
- Update the contact email by replacing `projectpstg@gmail.com` in the page.
- Replace `images/Logo.png` to update the navigation, hero, footer, and browser favicon logo.
- The visual theme is controlled by the CSS variables near the top of `index.html`.
